## About this example

This example shows how to run and compile a simple CUDA program.

## Requirements

CUDA Toolkit and proper Drivers.

## Run

Open a terminal and type:

```bash
sh run.sh
```


## Output

A typical output should look like this one. 

```
Hello World!

```
